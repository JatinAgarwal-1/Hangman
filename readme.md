# Hangman

Hangman is a word game.

## Run Locally

Clone the project

```bash
  git clone https://github.com/JatinAgarwal-1/Hangman.git
```

Go to the project directory

```bash
  cd Hangman
```

Install dependencies
```bash
pip install requests
```
